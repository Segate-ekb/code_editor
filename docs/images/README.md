# Изображения для документации

В эту папку нужно добавить скриншоты:

- `editor-preview.png` — превью редактора для главной страницы
- `autocomplete.png` — пример автодополнения
- `variables-panel.png` — панель переменных
- `themes.png` — примеры тем
- `snippets.gif` — анимация работы сниппетов

## Рекомендуемые размеры

- Превью: 800x500 px
- Скриншоты: 600-1200 px по ширине
- GIF-анимации: до 800 px по ширине
